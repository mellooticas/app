# Ícones PWA - Para gerar ícones PNG corretos:

1. Abra o arquivo generate-icons.html no navegador
2. Os ícones aparecerão na tela
3. Clique com botão direito em cada canvas e "Salvar imagem como..."
4. Salve como pwa-192x192.png e pwa-512x512.png

Ou use uma ferramenta online como:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

Upload os arquivos SVG para gerar os PNGs corretos.
